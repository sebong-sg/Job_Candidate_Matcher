3. Cosine Similarity Calculation
The semantic score is computed using cosine similarity between the two vectors:
def calculate_similarity(self, text1, text2):
    embedding1 = self.encode_text(text1)  # [0.123, -0.045, 0.287, ...]
    embedding2 = self.encode_text(text2)  # [0.115, -0.038, 0.301, ...]
    
    similarity = cosine_similarity([embedding1], [embedding2])[0][0]
    return float(similarity)

    cosine_similarity(A, B) = (A · B) / (||A|| * ||B||)

    Where:
A · B = dot product of vectors A and B
||A|| = Euclidean norm (magnitude) of vector A
||B|| = Euclidean norm (magnitude) of vector B

4. Step-by-Step Calculation Example:
Let's simulate with simplified 3D vectors:

Candidate Vector: [0.7, 0.5, 0.3]

A · B = (0.8 * 0.7) + (0.6 * 0.5) + (0.1 * 0.3)
      = 0.56 + 0.30 + 0.03

Step 2: Magnitude of Job Vector
||A|| = √(0.8² + 0.6² + 0.1²)

Step 4: Cosine Similarity
similarity = 0.89 / (1.005 * 0.911)
